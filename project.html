EDU CLOUD MANAGEMENT SYSTEM

student teacher interaction

  • Allows messaging and communication between students and teachers.
  • Teachers can post announcements and assignments
  • Students can ask questions and receive feedback.

attendance tracking

  • Teachers can mark daily attendance of students in each class.
  • Students can view their attendance records.
  • Admin can generate monthly/yearly attendance reports.

Student Rank Level Tracking

  • System calculates rank based on test scores and overall performance.
  • Ranks are displayed department-wise or batch-wise.
  • Graphs or progress bars show academic trends.

Course Suggestions

  • Suggests courses based on student performance and interest areas.
  • Uses past results and preferences to recommend new subjects.

Technologies used

  • Backend: Django (Python)
  • Frontend: HTML, CSS, JavaScript
  • Database: SQL (e.g., SQLite/PostgreSQL/MySQL)
